# 鼠标停留时播放

移动鼠标到GIF区域则播放，离开则停止播放

```md
![](charlie.gif)
```

![](charlie.gif)

# 鼠标点击后播放

鼠标点击播放按钮就播放，否则不播放

```md
![](charlie.gif "-gifcontrol-mode=click;")
```

![](charlie.gif "-gifcontrol-mode=click;")

# 自动循环播放

```md
![](charlie.gif "-gifcontrol-disabled;")
```

![](charlie.gif "-gifcontrol-disabled;")
